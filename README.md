#### Statistical-Learning
This repository consists of labs in the book: [An Introduction to Statistical Learning](https://goo.gl/TSMkhu).  

Labs covered:  
* Linear Regression    
* Model Regularization  
* Logistic Regression  
* K-Nearst-Neighbors  
* Cross-Validation    

Labs to be covered:  
* Tree-Based Methods  
* Support Vector Machines  
* Unsupervised Learning
